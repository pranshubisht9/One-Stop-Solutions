## Tech Stack Used

### Front End
* Javascript
* HTML
* CSS

### Back End
* Java-8
* Spring Boot
* Maven
* Postman
* Swagger
* Validation
* Lombok

## Module Wise Functionality

### Customer
* Login/Registration
* View Issue
* Create/ Delete Issue
* Reopen issue

### Operator
* Login
* View Issue by id/ type
* View Customer by Firstname/Email/Id
* Create Solution
* View Solurtion by Id, Issue Id.

### Admin 
* Login/Register
* Create Operator
* Create Department
* View Department
* View Employee
